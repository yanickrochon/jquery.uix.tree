jQuery UIx Tree
==================

This is a proof of concept implementation of a Tree component using jQuery + jQuery UI widget factory. 
It is jQuery UI theme enabled.

####**IT IS NOT READY FOR PRODUCTION USE!**

Every contribution is walcome!
